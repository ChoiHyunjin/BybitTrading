## 상태 정의

1. **idle**: 초기 상태, 설정만 보임
2. **loading**: "백테스트 실행 중..." + 프로그레스
3. **success**: 결과 요약 + 거래 내역 표시
4. **error**: 에러 메시지 표시

## 인터랙션

- 실행 버튼 탭 → loading → 데이터 fetch → 엔진 실행 → success
- 에러 시 재시도 버튼 표시
- 거래 내역 스크롤 가능

## UX Copy

- 실행 버튼: "백테스트 실행"
- 로딩: "데이터를 불러오는 중..." → "전략을 실행하는 중..."
- 에러: "데이터를 불러올 수 없습니다. 네트워크 연결을 확인해주세요."
- 빈 결과: "해당 기간에 거래 신호가 발생하지 않았습니다."
